하이카서비스 특약 가입고객님을 위한
고장출동서비스입니다.

서비스 내용

01배터리 방전 시 시동
서비스 내용
배터리 방전(차량 키 배터리, 연료전지 등은 대상이 아님)으로 시동을 걸 수 없는 경우 시동 조치
추가 비용
새 배터리로 교체 시
02전기차 방전 시 응급지원
서비스 내용
고전압 배터리가 방전되어 자력으로 운행할 수 없거나, 계기판에 표시된 주행가능거리가 10km 이내인 경우 하루 1회에 한하여 충전소까지 60km를 한도로 견인(2021.04.06 책임개시 계약부터 가까운 충전소까지 100km 한도로 견인)
서비스 제한
4륜차량 타이어 잠금해제 불가능한 경우
추가 비용
초과 1km당 2,200원
03타이어 펑크 수리
서비스 내용
피보험자동차 지면에 닿는 타이어트레드 부분(접지면)의 구멍이 뚫린 단순 펑크로 운행할 수 없는 경우 펑크 1개 부위 수리
서비스 제한
야간, 기상상황 등으로 인해 펑크위치를 육안으로 확인하기 어렵거나 특수장비가 필요한 타이어
추가 비용
초과 타이어 펑크 1개 초과 수리 시
04타이어 교체
서비스 내용
타이어 펑크 수리가 불가능한 경우 피보험자동차에 내장된 예비 타이어로 교체
서비스 제한
자동차 제작사에서 출고 당시의 휠이 아닌 경우로, 해당 휠의 교체장비를 보유하고 있지 않을 경우
05잠금장체 해제
서비스 내용
자동차 키를 자동차 실내에 두고 차문을 잠근 경우 잠금장치 해제
서비스 제한
트렁크 잠금장치 제외 외제차에는 서비스를 제공하지 않음
06비상급유
서비스 내용
연료가 소진되어 운행이 불가능한 경우, 하루 1회에 한해 3리터 한도로 급유(휘발유, 경유에 한함)
서비스 제한
보험기간 중 2회만 제공, LPG차량에는 서비스를 제공하지 않음(가까운 충전소로 견인하는 서비스로 대체하여 제공)
추가 비용
휘발유, 경유 3리터 초과하여 급유 시
07긴급견인
서비스 내용
사고 또는 고장으로 자력 운행이 불가능한 경우 하루 1회에 한하여 피보험자가 요청하는 곳까지 10km 한도로 견인(전기차 또는 견인확장 특약 가입 시 60km까지 견인, 전기차는 2021.04.06 책임개시 계약부터 100km까지 견인)
서비스 제한
보험기간 중 4륜차량 타이어 잠금해제가 불가능한 경우
추가 비용
초과 1km당 2,200원
08긴급구난
서비스 내용
피보험자동차가 도로를 이탈하거나 장애물 등으로 자력 운행이 불가능한 경우 서비스 제공
추가 비용
별도 구난장비가 필요한 경우, 구난소요시간 30분 초과 시
09휴즈교환
서비스 내용
피보험자동차의 휴즈 단락으로 운행이 불가능한 경우 새 휴즈로 교환
추가 비용
엔진룸 릴레이박스의 휴즈 교환 시
10Road 차량진단
서비스 내용
피보험자동차 계기판에 경고등이 점등되거나 명확하지 않은 원인으로 시동을 걸 수 없는 경우 휴대용 진단기기를 활용하여 차량 이상유무 확인
서비스 제한
외제차 및 OBD1 시스템 장착 차량에는 서비스 제공 불가
OBD 1.2 차종 확인하기
하이카프라자 찾기

이용대상

  • 하이카서비스 특약 가입고객
    • 가입대상: 승용차 / 경·4종화물 / 경·3종승합
    • 제공횟수: 6회(2018.07.26 이전 책임개시일 발생 계약은 5회. 단, 단기계약 보험기간에 따라 제공횟수 다름. 3개월 미만 1회, 3개월 초과 1년 미만으로 3회 제공)
  • 하이카서비스 특약(중형 화물·승합) 가입고객
    • 가입대상:
      1. 2종승합 또는 2·3종 화물
      2. '기중기장치 자동차요율' 또는 '특수적재장치 자동차요율'이 적용되는 경·4종화물
      3. 구조변경 등으로 하이카서비스 특약 가입이 제한되는 3종승합, 경·4종 화물
      4. 그밖에 현대해상이 인정하는 특정용도 자동차 및 캠핑카
    • 제공횟수: 3회(단, 보험기간 1년 미만 1회 제공)
    • 제공서비스: 배터리 방전 시 시동서비스 / 잠금장치해제서비스 / 비상급유 / 긴급견인서비스에 한하여 제공

보험기간 중 서비스를 모두 이용한 경우에는 하이카서비스 특약만 별도로 가입할 수 없습니다.
