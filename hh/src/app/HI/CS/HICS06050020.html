계좌 1원 인증을 해 주세요.

고객님 명의 은행 계좌로 1원을 입금하면서 입금자명으로 인증단어를 보내드립니다.
입금내역을 바로 확인할 수 있도록 준비해 주세요.

은행/계좌번호(본인명의 계좌만 가능)

1원을 입금한 입금자명의 인증단어(한글 4글자)를 입력하고 [다음]을 눌러 주세요.
입금내역에서 1원 입금이 없을 경우, [인증 재요청]을 눌러서 다시 진행해 주세요.

이용안내
  1. 1(현대해상 화면) 인증 요청
  2. 2(은행) 입금내역의 입금자명 확인
  3. 인증(현대해상 화면) 입금자명 입력
  4. 4(현대해상 화면) 인증 완료
